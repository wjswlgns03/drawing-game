# 기능 요구사항

## 사용자 기능
1. **사용자 인증**:
   - Google, Facebook, Kakao를 통한 OAuth 로그인 지원.
   - 안전한 사용자 계정 관리.
2. **게임 참여**:
   - 기존 게임에 참여하거나 새로운 방 생성.
   - 실시간 그림 그리기 및 맞추기 기능.
3. **게임 모드**:
   - 기본 모드: 턴제 그림 그리기 및 맞추기.
   - 팀 모드: 팀원과 협업.
   - 테마 모드: 사전에 정의된 테마를 기반으로 맞추기.
   - 타임 어택 모드: 시간 제한 경쟁.

## 게임 메커니즘
1. **실시간 상호작용**:
   - 그림 캔버스 및 정답을 실시간으로 동기화.
2. **게임 상태 관리**:
   - 활성 사용자, 게임 라운드, 점수를 유지.
3. **공정한 플레이 정책**:
   - 방해 플레이어를 제거하기 위한 투표 시스템.
   - 부정행위 방지 메커니즘.
